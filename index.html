IMaint

IMaint is a piece of software used to improve maintenance management in property management companies. It is designed to keep track of work orders, monitor assets condition, manage inventory, improve billing and closing.

Classes

The current version contains the following classes:

Appliance

is a superclass for appliances. The class creates on object of appliances and takes manufacturer, serial number, and model number information.

public class Appliance
{
    private String model;
    private String serial;
    private String manufacturer;

    /**
     * Constructor for objects of class Appliance used when no info about an appliance is provided
     */
    public Appliance(){ }

    /**
     * Constructor for objects of class Appliance used when the initial information is provided
     */
    public Appliance (String manufacturer, String model, String serial)
    {
        this.model=model;
        this.serial=serial;
        this.setManufacturer(manufacturer);
    }

    /**
     * The method adds Serial Number, Model Number and Manufacturer Name to the object.
     * @param String manufacturer - manufacturer name (Bosch, GE, etc.).
     * @return nothing.
     * 
     */
    public void setAttributes (String manufacturer, String model, String serial)

    {
        this.model=model;
        this.serial=serial;
        this.setManufacturer(manufacturer);
    }

    /**
     * A setter for Model Number.
     * @param Model Number as a string.
     * @return nothing.
     */
    public void setModelNumber(String model)
    {
        this.model=model;       
    }

    /**
     * A getter for Model Number.
     * @return Model Number.
     */
    public String getModelNumber()
    {
        return model;
    }

    /**
     * A setter for Serial Number.
     * @param Serial Number as a string.
     * @return nothing.
     */
    public void setSerialNumber(String serial)
    {
        this.serial=serial;     
    }

    /**
     * A getter for Serial Number.
     * @return Serial Number.
     */
    public String getSerialNumber()
    {
        return serial;
    }

    /**
     * A setter for Manufacturer Name.
     * @param Manufacturer Name as a string.
     * @return nothing.
     */
    public void setManufacturer(String manufacturer)
    {
        this.manufacturer=manufacturer;     
    }

    /**
     * A getter for Manufacturer Name.
     * @return Manufacturer Name.
     */
    public String getManufacturer() {
        return manufacturer;
    }


}
